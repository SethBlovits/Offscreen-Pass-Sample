# Offscreen-Pass-Sample
Sample program that creates and offscreen pass to render into a render texture and then displayed into an ImGui UI.

Sample program showing rendering into a render texture with an offscreen pass and then displaying that with an ImGui window.
Uses some of my other header utilities to open a window and draw things.
